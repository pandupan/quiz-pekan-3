# 📝 Quiz Pekan 3

Proyek **kuiz mingguan** sebagai bagian dari program bootcamp atau pembelajaran. Berisi implementasi soal-soal dan fitur interaktif.

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
